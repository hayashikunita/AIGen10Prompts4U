## 概要

AIGen10Prompts4Uは、様々なシーンで使用できるシステムプロンプトを簡単に生成できるツールです。
1,120個以上のサンプルプロンプトから、用途に応じて選択したり、OpenAI APIを使って新しいプロンプトを生成することができます。

### 主な機能

- ✨ **サンプルからの抽出**: 1,120個以上の事前準備されたプロンプトから選択可能
- 🤖 **AI生成**: OpenAI GPT-5を使用してカスタムテーマのプロンプトを生成
- 💬 **チャットボット機能**: 選択したプロンプトでAIと会話、ファイル添付対応
- 📁 **豊富なカテゴリ**: 28カテゴリ、法律、経営、金融、QOL向上など多様な専門分野
- 📎 **添付ファイル推奨**: 各プロンプトに最適な添付ファイルの種類を提案
- 💾 **JSON出力**: 生成結果をJSON形式で保存し、再利用可能

## インストール

### 必要な環境

- Python 3.8以上

### セットアップ

1. リポジトリをクローン

```bash
git clone <repository-url>
cd AIGen10Prompts4U
```

2. 必要なパッケージをインストール

```bash
pip install -r requirements.txt
```

### 依存パッケージ

- **Streamlit**: Web UIフレームワーク
- **OpenAI**: ChatGPT API連携
- **pandas**: データ処理（Excel/CSV読み込み）
- **openpyxl**: Excelファイル処理
- **pdfplumber**: PDFファイルからのテキスト抽出
- **python-docx**: Wordファイル処理
- **python-dotenv**: 環境変数管理

3. (オプション) OpenAI APIを使用する場合

`.env`ファイルを作成し、APIキーを設定:

```bash
cp .env.example .env
# .envファイルを編集してAPIキーを設定
```

## 使い方

### Streamlit Web UIで使う（推奨）

1. **Streamlitアプリを起動**:
```bash
streamlit run streamlit_app.py
```

2. ブラウザが自動的に開き、WebUIが表示されます

3. **モードを選択**:
   - 🎲 **プロンプト生成モード**: カテゴリから複数のプロンプトを生成
   - 💬 **チャットボットモード**: 選択したプロンプトを使って会話

#### プロンプト生成モード
1. サイドバーでカテゴリと生成数を選択
2. 「プロンプト生成」ボタンをクリック
3. 生成されたプロンプトを確認・コピー
4. 「このプロンプトでチャット」ボタンでチャットボットモードに移行可能

#### チャットボットモード
1. 「プロンプトから選択」でシステムプロンプトを設定
2. **📎 ファイルを添付**（複数可）
   - **対応形式**: PDF (.pdf)、Word (.docx)、Excel (.xlsx, .xls)、CSV (.csv)、テキストファイル、コードなど
   - **PDFファイル**: ページごとにテキストを抽出して分析可能
   - **Wordファイル**: 文書の段落を抽出してAIが理解
   - **Excelファイル**: 全シートのデータを表形式でテキスト化
   - **CSVファイル**: データを読みやすい形式に自動変換
   - **テキストファイル**: コード、ドキュメント等を直接分析
   - **⚠️ ファイルサイズ制限**: 1ファイルあたり約60KB（15,000トークン）まで。大きいファイルは自動的に切り詰められます
3. チャット入力欄にメッセージを入力
4. AIが添付ファイルの内容を含めてシステムプロンプトに従って応答
5. 会話を継続できます
6. 「新しい会話」で会話をリセット

**注意**: チャットボット機能を使用するには、`.env`ファイルに`OPENAI_API_KEY`を設定する必要があります。

##### 📊 トークン制限について

OpenAI APIには1分あたりのトークン数制限があります（30,000トークン/分）。本アプリケーションでは以下の対策を実装しています:

- **自動切り詰め**: 各ファイルは最大15,000トークン（約60KB）まで自動制限
- **3段階警告システム**:
  - ✅ **〜20,000トークン**: 正常処理
  - ⚠️ **20,000〜24,999トークン**: 警告表示 + 処理続行
  - ❌ **25,000トークン以上**: エラー表示 + 処理中止

**大きなファイルを扱う際の推奨対応**:
- PDFファイル: 必要なページのみ抽出、または分割
- Excelファイル: 関連シートのみ別ファイルで保存
- 複数ファイル: 一度に2〜3ファイルまでに抑える
- ログファイル: エラー箇所を含む関連部分のみコピー

### コマンドラインで使う

#### 基本的な使い方

1. **カテゴリ一覧を表示**

```bash
python src/main.py --list
```

2. **サンプルから10個のプロンプトを生成**

```bash
python src/main.py --category industry
```

3. **生成数を指定**

```bash
python src/main.py --category sales --count 5
```

4. **画面表示をスキップしてファイルのみ出力**

```bash
python src/main.py --category management --no-display
```

5. **出力ファイル名を指定**

```bash
python src/main.py --category idea --output my_prompts.json
```

### OpenAI APIを使用した生成

```bash
python src/main.py --use-openai --theme "スタートアップのマーケティング戦略" --count 10
```

環境変数でAPIキーを設定していない場合:

```bash
python src/main.py --use-openai --theme "データ分析アドバイザー" --api-key YOUR_API_KEY
```

## トラブルシューティング

### エラー: プロンプトファイルが見つかりません

→ `prompts_data/`ディレクトリにJSONファイルが存在することを確認してください。

### エラー: OpenAI API Keyが設定されていません

→ `.env`ファイルを作成し、`OPENAI_API_KEY`を設定してください。

### モジュールが見つかりません

→ `pip install -r requirements.txt`を実行してください。

### PDFファイルからテキストが抽出できない

→ スキャンされた画像PDFはテキスト抽出できません。OCR処理が必要です。

### Excelファイルが読み込めない

→ `.xls`形式は`.xlsx`形式に変換してください。新しい形式での保存を推奨します。

### ファイルアップロード後、エラーメッセージが表示される

→ ファイルのエンコーディングを確認してください。UTF-8、Shift-JIS、CP932に対応しています。

### エラー: Request too large / rate_limit_exceeded (429エラー)

このエラーは、OpenAI APIのトークン制限（30,000トークン/分）を超えた場合に発生します。

**対処方法**:
1. **ファイルサイズを減らす**
   - PDFファイル: 必要なページのみ抽出（10ページ以内推奨）
   - Excelファイル: 大きなシートは行数を減らす、複数シートは分割
   - テキストファイル: 関連部分のみをコピーして新規ファイル作成

2. **アップロードファイル数を減らす**
   - 一度に2〜3ファイルまでに抑える
   - 大きなファイルの場合は1ファイルずつ分析

3. **警告が表示された場合**
   - ⚠️ **切り詰め警告**: ファイルの一部が省略されています。重要な部分が含まれているか確認
   - ⚠️ **20,000トークン警告**: 処理は続行されますが、応答に時間がかかる可能性があります
   - ❌ **25,000トークン超過**: ファイルを分割してください

4. **時間を空けて再試行**
   - レート制限は1分単位でリセットされます
   - 1〜2分待ってから再度試してください
